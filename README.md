# 🏠 Deal.pk - Pakistan's #1 Real Estate Platform

A modern, full-featured real estate marketplace built with Next.js 14, TypeScript, and Tailwind CSS.

## ✨ Features

- 🔍 Advanced property search with filters
- 🏷️ Premium and Featured listings
- 🗺️ Interactive map view
- 💳 Secure payments with RapidGateway
- 📱 Mobile-first responsive design
- 🌙 Dark/Light mode toggle
- 👤 Multi-role authentication (User, Agent, Admin)
- 📊 Admin dashboard with analytics
- 💼 Agent dashboard for property management
- ❤️ Wishlist and search alerts
- 📝 Blog and area guides
- 🎥 Video property tours
- 📐 Mortgage calculator
- 🔔 Real-time notifications
